# ♻️ SCRAPDEAL – Scrap Selling Platform

## 📌 Project Overview

SCRAPDEAL is a web-based platform that allows users to sell their scrap materials easily and efficiently. The system connects users with scrap collectors, making the process of recycling simple, organized, and eco-friendly.

This project is designed to promote sustainable waste management by providing a digital solution for scrap selling.

---

## 🚀 Features

### 👤 User Panel

* User Registration & Login
* Sell Scrap Request Submission
* View Order Status
* Manage Profile

### 🛠️ Admin Panel

* Admin Login
* Manage Users
* View Scrap Requests
* Update Order Status
* Manage Scrap Categories

## ⚙️ Installation & Setup

1. Clone the repository:

```
https://github.com/suvedant/Scrapdeal.git
```

2. Move project to XAMPP htdocs folder:

```
C:\xampp\htdocs\
```

3. Start XAMPP:

* Apache ✔️
* MySQL ✔️

4. Configure database:

* Import database file (if available)
* Update DB connection in config file

5. Run the project:

```
http://localhost/SCRAPDEAL/
```
